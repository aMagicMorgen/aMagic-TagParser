# Документация класса TagParser

## Общее описание

Класс `TagParser` предназначен для парсинга и генерации HTML-кода из упрощенного синтаксиса. Поддерживает вложенность тегов, PHP-блоки, Markdown, шаблонизацию и работу с переменными через сессии.

## Основные возможности

1. **Упрощенный синтаксис тегов** - сокращенная запись HTML тегов с атрибутами
2. **Вложенность тегов** - автоматическое управление открывающими и закрывающими тегами
3. **PHP-блоки** - выполнение PHP кода внутри шаблонов
4. **Markdown поддержка** - встраивание Markdown через zero-md
5. **Шаблонизация** - подстановка данных в шаблоны с использованием плейсхолдеров
6. **VAR блоки** - извлечение переменных из PHP кода в сессию
7. **Include** - подключение внешних файлов

## Константы

### `VOID_TAGS`
Список самозакрывающихся HTML тегов:
```php
const VOID_TAGS = ['area', 'base', 'br', 'col', 'embed', 'hr', 'img', 'input', 'link', 'meta', 'param', 'source', 'track', 'wbr'];
```

### `HEAD`
Мета-теги, автоматически вставляемые после открывающего тега `<head>`:
```php
const HEAD = '
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">';
```

### `ZERO_MD`
Скрипт для поддержки Markdown, вставляемый перед закрывающим тегом `</head>`:
```php
const ZERO_MD = "\n\t<script type='module' src='https://cdn.jsdelivr.net/gh/zerodevx/zero-md@2/dist/zero-md.min.js'></script>\n\t";
```

### `ZERO_MD_START` и `ZERO_MD_END`
Маркеры начала и конца Markdown блока:
```php
const ZERO_MD_START = "\n<zero-md>\n<script type='text/markdown'>\n";
const ZERO_MD_END = "</script>\n</zero-md>";
```

### `JQUERY`
Скрипт jQuery, вставляемый после закрывающего тега `</footer>`:
```php
const JQUERY = "\t<script src='https://code.jquery.com/jquery-3.6.0.min.js'></script>\n";
```

## Синтаксис

### Базовый синтаксис тегов

Формат строки:
```
[тег][#id][/name][.класс1.класс2][(атрибуты)] [текст]
```

#### Примеры:

**Простой тег:**
```php
div
// Результат: <div></div>
```

**Тег с классом:**
```php
p.text
// Результат: <p class="text"></p>
```

**Тег с ID:**
```php
div#header
// Результат: <div id="header"></div>
```

**Тег с именем:**
```php
input/username
// Результат: <input name="username">
```

**Тег с атрибутами:**
```php
a(href='/') Ссылка
// Результат: <a href='/'>Ссылка</a>
```

**Комбинированный пример:**
```php
ul#menu/main-menu.nav.navbar
// Результат: <ul id="menu" name="main-menu" class="nav navbar"></ul>
```

### Вложенность

Вложенность определяется отступами (пробелами или табуляцией):

```php
div.container
    h1 Заголовок
    p.paragraph Текст параграфа
        a(href='/') Ссылка
```

Результат:
```html
<div class="container">
    <h1>Заголовок</h1>
    <p class="paragraph">Текст параграфа
        <a href='/'>Ссылка</a>
    </p>
</div>
```

### PHP блоки

Используйте теги `PHP/` и `/PHP` для вставки PHP кода:

```php
PHP/
<?php
    $name = "Мир";
    echo "<h1>Привет, $name!</h1>";
?>
/PHP
```

### Markdown блоки

Используйте тег `MD/` для начала Markdown блока:

```php
MD/
# Заголовок Markdown
- Пункт 1
- Пункт 2
/MD
```

### Include

Подключение внешних файлов:

```php
include ./header.php
```

### Переменные и шаблоны

#### Формат `{переменная}`

Замена переменной с сохранением форматирования:

```php
{title}Заголовок страницы!!!
<h1>{title}</h1>
```

#### Формат `[переменная]`

Замена переменной с добавлением перевода строки:

```php
[title]Заголовок!!!
<div>[title]</div>
```

#### Формат `$переменная`

Замена переменной с удалением переводов строк:

```php
$textМногострочный текст!!!
<p>$text</p>
```

### VAR блоки

Извлечение переменных из PHP кода в сессию:

```php
VAR/
    $user = "Иван";
    $email = "ivan@example.com";
/VAR

// После выполнения переменные будут доступны в $_SESSION['VAR']
```

### Шаблоны с данными

Формат для генерации повторяющихся блоков:

```
[Заголовок1|Заголовок2]
[Значение1|Значение2]
[Значение3|Значение4]
/% 
Шаблон с {Заголовок1} и {Заголовок2}
```

## Методы класса

### `__construct(string $input)`

Конструктор класса. Принимает строку с шаблоном или путь к файлу.

**Параметры:**
- `$input` - строка с HTML-шаблоном или путь к файлу

**Пример:**
```php
$parser = new TagParser("./template.tpl");
echo $parser; // Выведет сгенерированный HTML
```

### `__toString(): string`

Магический метод, возвращающий сгенерированный HTML.

**Пример:**
```php
$parser = new TagParser("h1 Заголовок");
echo $parser; // <h1>Заголовок</h1>
```

## Внутренние методы (приватные)

### `isVoidTag(string $tag): bool`
Проверяет, является ли тег самозакрывающимся.

### `getTemplate(string $input)`
Получает содержимое шаблона из файла или возвращает строку. Удаляет многострочные комментарии.

### `filterEmptyLines($lines)`
Удаляет пустые строки из массива.

### `mdsBlock(string $input)`
Обрабатывает Markdown блоки с заменой переменных.

### `parseLine(string $line): array`
Парсит строку в формате тег#id/name.класс(атрибуты) текст и возвращает массив с узлом.

**Возвращает:**
```php
[
    'tag' => 'div',
    'attr' => ['id' => 'main', 'name' => '', 'class' => 'container'],
    'attrs' => 'src="/image.jpg"',
    'text' => 'Текст'
]
```

### `phpParser(string $phpkod): string`
Выполняет PHP код и возвращает результат.

### `renderTag(array $node): string`
Рендерит HTML тег из массива узла.

### `parseInputToArray($input)`
Преобразует строку с данными в ассоциативный массив для шаблонизации.

### `applyTemplate($template, $record)`
Заменяет плейсхолдеры в шаблоне на значения из записи.

### `parseShablon($input)`
Парсит шаблон с данными и возвращает результат.

### `extractVarBlocks($text)`
Извлекает VAR блоки из текста и сохраняет переменные в сессию.

## Примеры использования

### Пример 1: Простая страница

```php
$template = "
html(lang='ru')
    head
        title Мой сайт
    body
        div.container
            h1.main-title Добро пожаловать!
            p.text-content Это пример использования TagParser
";

$parser = new TagParser($template);
echo $parser;
```

Результат:
```html
<!DOCTYPE html>
<html lang='ru'>
<head>
    <title>Мой сайт</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
    <div class="container">
        <h1 class="main-title">Добро пожаловать!</h1>
        <p class="text-content">Это пример использования TagParser</p>
    </div>
</body>
</html>
```

### Пример 2: Использование PHP

```php
$template = "
html
    body
        PHP/
            \$users = ['Иван', 'Петр', 'Сидор'];
            foreach(\$users as \$user) {
                echo \"<li>\$user</li>\";
            }
        /PHP
";

$parser = new TagParser($template);
echo $parser;
```

### Пример 3: Генерация списка из данных

$template = "

### Пример 4: Markdown и jQuery

## Особенности и примечания

1. **Автоматическое добавление DOCTYPE** - при использовании тега `html` автоматически добавляется `<!DOCTYPE html>`

2. **Автоматическое добавление мета-тегов** - при использовании тега `head` автоматически добавляются мета-теги из константы `HEAD`

3. **Автоматическое подключение jQuery** - при использовании тега `footer` автоматически добавляется скрипт jQuery

4. **Автоматическое подключение ZeroMD** - при использовании тега `head` перед его закрытием добавляется скрипт для поддержки Markdown

5. **Комментарии** - строки с обратным слешем `\` преобразуются в HTML комментарии

6. **Удаление многострочных комментариев** - все комментарии вида `/* ... */` автоматически удаляются из шаблонов

7. **Работа с сессиями** - для использования VAR блоков необходимо предварительно вызвать `session_start()`

## Обработка ошибок

Класс обрабатывает следующие ошибки:

- **Ошибки парсинга PHP** - перехватываются и выводят сообщение
- **Отсутствие файлов** - если файл не найден, возвращается исходная строка
- **Необработанные переменные** - фигурные скобки без значений удаляются
